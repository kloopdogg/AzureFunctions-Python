# AzureFunctions-Python
Sample Azure Functions project with Python

## Project Structure
The recommended folder structure for a Python functions project looks like the following example:
```text
<project_root>/
 | - .venv/
 | - .vscode/
 | - function_app.py
 | - additional_functions.py
 | - tests/
 | | - test_my_function.py
 | - .funcignore
 | - host.json
 | - local.settings.json
 | - requirements.txt
 | - Dockerfile
```
